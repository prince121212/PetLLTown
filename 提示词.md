# 宠物视频生成提示词

本文档用于根据一张宠物参考照片，生成项目第一阶段需要的 3 条基础透明宠物视频。

统一要求：

```text
根据我上传的宠物参考照片生成视频。保留宠物的品种、毛色、脸型、眼睛、耳朵、身体比例和整体可爱感。
视频为单只宠物主体，纯绿色背景，绿幕，无阴影，无纹理，便于后期抠图。
宠物完整入镜，不裁耳朵、尾巴、脚。宠物居中。
镜头固定，不推拉，不摇镜，不旋转。画面稳定，不要闪烁，不要变形，不要出现多余肢体。
输出竖版 3:4，720x960，24fps，无背景音乐。
```

## 1. 待机视频 idle

```text
根据我上传的宠物参考照片，生成一条 5 秒的宠物待机视频。

动作要求：
宠物保持自然站立或坐姿，整体安静可爱，做轻微呼吸、自然眨眼、耳朵轻动、尾巴轻轻摆动等细小动作。动作幅度要小，不能夸张，不要跳跃，不要转身，不要走出原位。

技术要求：
纯绿色背景，绿幕，无阴影，无纹理，便于后期抠图；720x960；24fps；5 秒；轻微呼吸声；宠物完整入镜；固定镜头；宠物位置稳定。
```

## 2. 倾听视频 listening

```text
根据我上传的宠物参考照片，生成一条 4 秒的宠物倾听视频。

动作要求：
宠物像是在认真听用户说话，轻轻抬头或看向镜头，耳朵有细微反应，身体可以轻微前倾，表情专注、温柔、亲近。动作自然克制，不要夸张张嘴，不要大幅移动，不要跳跃，不要离开原位。

技术要求：
纯绿色背景，绿幕，无阴影，无纹理，便于后期抠图；720x960；24fps；4 秒；轻微呼吸声；宠物完整入镜；固定镜头；
```

## 3. 回应视频 reply

```text
根据我上传的宠物参考照片，生成一条 4 秒的宠物回应视频。

动作要求：
宠物像是在开心回应用户，可以轻轻点头、眨眼、微微张嘴、尾巴轻摆或做一个小幅开心动作。情绪要友好、治愈、可爱，不要过度兴奋，不要跳跃，不要转圈，不要离开原位。

技术要求：
纯绿色背景，绿幕，无阴影，无纹理，便于后期抠图；720x960；24fps；4 秒；轻微呼吸声；宠物完整入镜；固定镜头；
```
